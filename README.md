# HR_Analytics
Power Bi Dashboard
Project Objective= To analyze employee performance, attrition, and workforce distribution.
Helps HR make data-driven decisions to improve retention.

Insights= Majority attrition happens in employees with low experience & low salary.
Sales & Operations departments have higher employee churn compared to others.

Steps= Imported HR dataset → cleaned null values → created DAX measures for attrition %.
Built charts for age, department, salary bands & job satisfaction.

Action Items= Increase retention benefits for low-experience employees.
Focus training & engagement programs in Sales and Operations teams.
